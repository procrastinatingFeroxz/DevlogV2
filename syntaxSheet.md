| Concept  | Python               | JavaScript                 |
| -------- | -------------------- | -------------------------- |
| Print    | `print("hi")`        | `console.log("hi")`        |
| Variable | `x = 5`              | `let x = 5`                |
| Constant | —                    | `const x = 5`              |
| Function | `def f():`           | `function f() {}`          |
| If       | `if x > 5:`          | `if (x > 5) {}`            |
| Loop     | `for i in range(5):` | `for (let i=0;i<5;i++) {}` |
| List     | `[1,2,3]`            | `[1,2,3]`                  |
| Dict     | `{"a":1}`            | `{a:1}`                    |




| Type      | Example         |
| --------- | --------------- |
| Number    | `42`, `3.14`    |
| String    | `"hello"`       |
| Boolean   | `true`, `false` |
| Undefined | `let x;`        |
| Null      | `null`          |
| Object    | `{a: 1}`        |
| Array     | `[1,2,3]`       |


> **notes:**
